# Chatbot CLI powered by 

Simple Chatbot CLI to test OpenAI API

## Getting Started

### Installing

* You will need rename `.env.base` and rename to `.env` also add your OpenAI API KEY, replacing `YOUR_KEY_HERE`
* After that, use:
```
npm i
```

### Executing program

```
npm start
```
